# 💰 ExpenseMind AI

An AI-powered personal finance assistant that helps users track expenses, manage budgets, analyze spending habits, and receive intelligent financial insights using Generative AI.

---

## 🚀 Overview

ExpenseMind AI is a modern full-stack application built to simplify personal finance management. It combines traditional expense tracking with AI-powered analysis, enabling users to understand their spending patterns and make smarter financial decisions.

---

## ✨ Features

### 🔐 Authentication
- Secure User Registration
- JWT-based Login
- Password Hashing
- Protected Routes

### 💸 Expense Management
- Add Expenses
- Edit Expenses
- Delete Expenses
- Categorize Expenses
- Search & Filter Expenses

### 📊 Dashboard & Analytics
- Monthly Expense Summary
- Category-wise Spending
- Budget Tracking
- Interactive Charts
- Financial Reports

### 🤖 AI Assistant
- AI-powered Spending Analysis
- Budget Recommendations
- Monthly Financial Summary
- Natural Language Expense Queries

### 📄 Future Enhancements
- PDF Bank Statement Upload
- OCR-based Expense Extraction
- RAG-powered Financial Assistant
- Expense Prediction
- Multi-Currency Support
